# Project Screenshots

This folder is reserved for screenshots used in the main README.

## Shot List (capture these)

1. GUI – Sequences tab
   - Sidebar with sequences list
   - Step editor with actions visible (e.g., move_to + click)
   - Group Call dropdown visible (if applicable)

2. GUI – Failsafe tab
   - Failsafe enabled; template selected; confidence spinner
   - Region label populated
   - Failsafe sequence with a normal step and a Group Call step

3. GUI – Templates tab
   - Template list and preview
   - Capture / Load Image buttons visible

4. Web – Dashboard (/)
   - Navigation links row
   - Token field with “Remember token” checkbox
   - Monitor selector and live MJPEG preview (ok if it’s “connecting…”) 

5. Web – Sequences (/static/sequences.html)
   - Sequence list, steps, and action editor
   - “Group” dropdown next to “Add Group Call”
   - Preview with drawn selection box

6. Web – Failsafe (/static/failsafe.html)
   - Failsafe settings (template + confidence)
   - Steps editor with action fields toggling (e.g., wait only shows seconds)
   - “Use Preview as Failsafe Region” button

7. Web – Groups (/static/groups.html)
   - Groups list, step editor, nested actions
   - Live preview and selection box

8. Web – Schedules (/static/schedules.html)
   - Schedule rows with Enabled, Sequence, Time

## Capture Tips

- Preferred window size: 1280×800 or 1600×900 for clarity.
- Use consistent dark/light theme across shots.
- Crop to focus on the panel; avoid full desktop unless showing multi-monitor preview.
- For web pages, include the header with navigation and token area.
