1. GUI – Sequences tab
   - Sidebar with sequences list
   - Step editor with actions visible (e.g., move_to + click)
   - Group Call dropdown visible (if applicable)

2. GUI – Failsafe tab
   - Failsafe enabled; template selected; confidence spinner
   - Region label populated

3. GUI – Templates tab
   - Template list and preview

4. Web – Dashboard (/)

8. Web – Schedules (/static/schedules.html)

## Capture Tips

- Preferred window size: 1280×800 or 1600×900 for clarity.
- Use consistent dark/light theme across shots.
- Crop to focus on the panel; avoid full desktop unless showing multi-monitor preview.
- For web pages, include the header with navigation and token area.
